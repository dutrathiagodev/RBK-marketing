## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML5 e CSS3
- JavaScript (ES6+)
- React
- Vite
- Git e GitHub
- Vercel

## 💻 Projeto

O **RBK Marketing** é uma landing page de alta performance voltada para a agência de marketing digital liderada por Ramiro Bilek. O objetivo da aplicação é apresentar os serviços, treinamentos, cases de sucesso e capturar leads de forma estratégica.

- [Acesse o projeto finalizado, online](https://rbk-marketing.vercel.app)

## ✨ Funcionalidades

- **Hero Section:** Apresentação de alto impacto visual e splash screen com a identidade da marca.
- **Métricas de Desempenho:** Indicadores de resultados (mais de R$ 1M+ em tráfego gerido e 240+ empresas impulsionadas).
- **Portfólio & Serviços:** Apresentação clara das soluções e treinamentos oferecidos.
- **Responsividade:** Layout adaptado para dispositivos móveis, tablets e desktops.
- **Conversão Direta:** Botões de CTA direcionados para atendimento e orçamentos.
